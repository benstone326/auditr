# auditr

Design-system audit app built with React + Vite.

## Getting started

```bash
npm install
npm run dev
```

## Common commands

```bash
npm run lint
npm run build
```

> If `node_modules` is missing, `predev`, `prelint`, and `prebuild` now auto-run `npm install` to prevent the `'<tool>' is not recognized` errors in fresh environments.
